# 푸드트럭 타이쿤 - 게임 기획서 (GDD)

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 클리커 + 방치형 타이쿤 |
| **테마** | 푸드트럭 운영 |
| **플랫폼** | 모바일 (세로 화면) |
| **엔진** | Unity 3D |
| **카메라** | 고정 (이동/회전 없음) |
| **조작** | 화면 터치 1종류 |

---

## 2. 핵심 루프

```
클릭 → 음식 제작 → 손님에게 판매 → 골드 획득 → 업그레이드 → 반복
                                         ↑
                        자동 생산 (요리사) ─┘
```

---

## 3. 재화

| 재화 | 획득 | 용도 |
|------|------|------|
| **골드** | 클릭 / 자동 생산 | 업그레이드 구매 |

---

## 4. 업그레이드 시스템

### 4-1. 클릭 업그레이드 (3종)

| 이름 | 비용 (Lv1/2/3) | 효과 | 비주얼 |
|------|----------------|------|--------|
| **메뉴 추가** | 100 / 500 / 2000 | 클릭당 +2 / +5 / +15G | 핫도그→타코→버거 메뉴판에 추가 |
| **고급 재료** | 150 / 600 / 2500 | 클릭 수익 x1.5 / x2 / x3 | 음식에 반짝이 이펙트 |
| **황금 손** | 200 / 800 / 3000 | 크리티컬 10% / 20% / 30% (2배) | "CRITICAL!" + 황금 코인 |

### 4-2. 자동 업그레이드 (3종)

| 이름 | 비용 (Lv1/2/3) | 효과 | 비주얼 |
|------|----------------|------|--------|
| **요리사 고용** | 100 / 500 / 2000 | 초당 +1 / +3 / +10G | 트럭 옆 요리사 등장 |
| **마케팅** | 200 / 800 / 3000 | 전체 수익 x1.2 / x1.5 / x2 | 손님 +1 / +2 / +3명 |
| **트럭 업그레이드** | 300 / 1200 / 5000 | 모든 수익 +10% / +25% / +50% | 트럭 외관 변경 |

---

## 5. 수익 계산 공식

```
[클릭 수익]
(1 + 메뉴추가) × 고급재료 × 크리티컬 × 마케팅 × 트럭

[자동 수익 (초당)]
요리사 × 마케팅 × 트럭
```

### 계산 예시 (모든 업그레이드 Lv2)

```
클릭: (1+5) × 2 × 1.5 × 1.25 = 22.5G
크리티컬 시: 22.5 × 2 = 45G
자동(초당): 3 × 1.5 × 1.25 = 5.6G
```

## 6. 성장 표현

| 단계 | 조건 | 변화 |
|------|------|------|
| **Stage 1** | 시작 | 낡은 트럭 / 핫도그 / 손님 1명 |
| **Stage 2** | 메뉴 Lv2 | 깨끗한 트럭 / 타코 추가 / 손님 3명 |
| **Stage 3** | 메뉴 Lv3 | 네온 트럭 / 버거 추가 / 손님 5명 |

---

## 7. 게임 필 (피드백)

### 7-1. 클릭 피드백

| 요소 | 연출 |
|------|------|
| **음식 팝업** | 해금된 최고 메뉴 아이콘이 위로 튀어오름 |
| **골드 표시** | "+22G" 텍스트가 떠오르며 페이드아웃 |
| **트럭 반응** | 살짝 흔들림 (Punch Scale) |
| **파티클** | 연기/김 이펙트 |

### 7-2. 사운드

| 상황 | 효과음 |
|------|--------|
| 클릭 | "치익~" 조리 소리 |
| 골드 획득 | "챠링~" 코인 소리 |
| 크리티컬 | "따단!" 강조음 |
| 업그레이드 | "레벨업!" 효과음 |

## 8. UI 레이아웃

```
┌─────────────────────────┐
│       골드: 12,345      │ ← 보유 골드
│      +5.6G/sec          │ ← 초당 자동 수익
│                         │
│  ┌─────────────────┐    │
│  │ MENU            │    │
│  │ [1] [2] [3] [ ] │    │ ← 메뉴판 (해금 현황)
│  └─────────────────┘    │
│                         │
│        [트럭]           │ ← 푸드트럭 (클릭 영역)
│       [요리사]          │ ← 요리사 (고용 시 표시)
│    [손님] [손님] ...    │ ← 손님 줄
│                         │
├─────────────────────────┤
│ 메뉴   │ 재료  │ 황금손  │ ← 클릭 업그레이드
│ Lv2    │ Lv1   │ Lv1    │
├─────────────────────────┤
│ 요리사 │마케팅 │ 트럭   │ ← 자동 업그레이드
│ Lv2    │ Lv1   │ Lv1    │
└─────────────────────────┘
```
